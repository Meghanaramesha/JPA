# How to analyze mobile trends in just a few minutes.

![](https://www.gamigion.com/wp-content/uploads/2024/11/How-to-analyze-mobile-trends-in-just-a-few-minutes.jpg)

###### How to analyze mobile trends in just a few minutes.

Here’s a step-by-step guide using ChatGPT and AppMagic:   
  
1. Go to AppMagic   
• Use [this link](https://appmagic.rocks/top-charts/trending?newOnly=1&tag=3&period=last365days&country=W1&promoCode=67488779e9b91&utm_source=AS) for temporary premium access.  
  
2. Find “Top Trending”   
• In the Apps section, choose “Top Trending.”   
  
3. Set the Right Tags   
• Choose “Gainers”   
• Select “Games”   
• Filter by “New Only”   
• Use Tier-1 West markets   
• Pick “Top Free” or “Top Grossing”   
  
4. Download the CSV   
• Click “Download CSV” and save the file to your device.   
  
5. Go to ChatGPT   
• Log in at [<https://chatgpt.com>].   
  
6. Upload the CSV   
• Attach the downloaded file in the ChatGPT text box.   
  
7. Use the Prompt   
• Grab the specific [prompt here](https://artistic-potassium-c0d.notion.site/Prompt-for-Analyzing-the-Top-100-Trending-Games-Dataset-13faa95b2cf88016bfeee3d027fa3904?pvs=4).  
  
8. Type “Continue”   
• The analysis takes time, so repeat “continue” until the results are complete.   
  
9. Get the Insights   
• Discover patterns, trends, genres, top publishers, and visual data like charts.   
  
This workflow helps uncover what’s driving success in mobile games quickly and efficiently.
