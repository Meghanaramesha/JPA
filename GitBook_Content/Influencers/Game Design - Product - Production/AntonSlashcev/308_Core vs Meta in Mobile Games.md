# Core vs. Meta in Mobile Games

**Here’s a breakdown of time distribution in different games:**

8. Caesars Casino: Core – 74.7%, Meta – 25.3%  
9. House of Fun: Core – 77.6%, Meta – 22.4%  
10. June’s Journey: Core – 78.1%, Meta – 21.9%

13. Toon Blast: Core – 81.9%, Meta – 18.1%  
14. Texas Poker: Pokerist: Core – 82.5%, Meta – 17.5%  
15. Bingo Blitz: Core – 83.9%, Meta – 16.1%  
16. Best Fiends: Core – 85.1%, Meta – 14.9%

18. Zynga Poker: Core – 88.0%, Meta – 12.0%
